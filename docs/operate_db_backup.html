Database Backup for Beginners

You need to do a forum backup, and yes you need do do it now, put it off and it may be to late!

  1. Login to your cPanel and go to your "phpMyAdmin" section.
  2. Select your database (if you have more than one).
  3. Once loaded, click on the "Export" tab at the top of the page.
  4. Under the Export table you will see a large box showing either the database name (if so just highlight), or a list of tables starting with phpbb_ , make sure the SQL input button is chosen, and click on "Select All" which will highlight all tables.
  5. Under the SQL Options table (in the bottom Data section) tick the "Complete inserts" checkbox.
  6. Under the Structure table (which should be auto-checked) you will see a "ADD DROP TABLE" checkbox which you should also tick.
  7. Near the bottom of the page tick the "Save as file" checkbox.
  8. If you would like to compress the file into a zipped/gzipped format select so, and click "Go".
  9. You will then get a normal popup save window, where you should select "Save to disk".

Easy! The database backup is created, the advantage is that you have the file on your hard-drive, and will be able to check if it's complete. If it's less than 8 MB, you know something has gone wrong, so just try again.

  1. Open your backup file in a text editor and scroll to the bottom.
  2. The last thing you should see is the "phpbb_words" table dumping data, if it is you know your backup is completely successful.
