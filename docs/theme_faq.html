Theme FAQ

This is a very basic Frequently Asked Questions (FAQ) section which attempts to answer some of the more commonly asked theme questions. Our thanks to all those people who've converted themes, beta tested, contributed bug reports and code fixes in the past.

  1. Template->make_filename(): Error - file does not exist!

    2. This error is usually caused by a template file that has not uploaded to your server correctly, and as such is missing or incomplete. This error can also be caused by trying to use a theme that is not compatible with your Fully Modded version. Re-upload the missing file again, or please check the themes that you are using are compatible with the version of Fully Modded phpBB that you are using.

  2. Can I install/use phpBB 2 themes?

    3. You can not use alternative phpBB templates for Fully Modded, you should note they will not function correctly with your Fully Modded phpBB board. It is therefore recommended you only use themes available on our website that are compatible with the version that you are using. Themes downloaded from 3rd party sites are not supported by us.
