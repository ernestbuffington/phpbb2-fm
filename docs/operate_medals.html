How do Medals work?

A complete medal system consisting of medal management panel, medal listing page to display all medal information, medal moderator management panel for adding/removing medals from users, e-mails are sent to users receiving medals, display amount of medals a user holds on post and profile pages.

  1. How to add/edit/delete medals?

    Go to Administration Panel -> Board Settings -> Medals -> Manage medals.

  2. How to add/delete medal moderators?

    Go to Administration Panel -> Board Settings -> Medals -> Manage medals. Select the Medal you wish to edit, click Medal Moderator.

  3. How to show medal images on posts?

    Go to Administration Panel -> Board Load Settings -> Choose Yes for "Display users' medals on posts".

  4. How to show random medal images on posts?

    Go to Administration Panel -> Board Settings -> Medals. Choose Yes for "Allow random medal image on posts".

  5. How to edit the number of medals shown on posts?

    Go to Administration Panel -> Board Settings -> Medals. Enter Rows x Columns.

  6. How to change the dimensions of medal images on posts?

    Go to Administration Panel -> Board Settings -> Medals. Enter Height x Width.

  7. How to award members with medals?

    Log in as Forum Admin or appointed as Medal Moderator. Go to http://youdomain/phpBB2/medals.php. Select the Medal, click on "Medal Control Panel". In "Username", type in the member's username. In "Award Reason", type in the reason. Click on Submit. You can award a user with the same Medal multiple times.

  8. How to remove medals from members?

    Log in as Forum Admin or appointed as Medal Moderator. Go to http://yourdomain/phpBB2/medals.php. Select the Medal, click on "Medal Control Panel". Under "Remove this Medal from one or more specific users". Choose the those you want to remove. Click on Submit. Each award is logged, so you know which one you want to remove.

  9. How to edit members' medal reason?

    Log in as Forum Admin or appointed as Medal Moderator. Go to http://yourdomain/phpBB2/medals.php. Select the Medal, click on "Medal Control Panel". In "Members having this medal", click on that member's username. Type in the new/edited reason. Click on Submit.
