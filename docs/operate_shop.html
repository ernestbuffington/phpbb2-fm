How do the Shops work?

In the shop the member can buy effects or other inputed items. The Administrator can switch the Effect Shop on and create additional shops from Admin -> Points System -> Shop: Configuration. In the Effect or Special Shop the member can buy privileges like using an avatar or a signature, and colors for their name and titles.The Administrator can add items to the additional shops with a name, long and short description, a price, a stock and a maximum stock.

  1. I just cant get item images to appear!

    This can be caused by a few things. First, the shop only allows .gif and .jpg images at the moment, all images must be placed in the images/shop/ directory. Next, make sure if they're not showing in profiles or on threads, that you have the options set to images and not links from Admin -> Points System -> Shop: Configuration.

    Lastly, one of the more common mistakes, make sure the images are named EXACTLY the same as the item (this is case sensitive). For example, apple must have an apple.jpg image (or .gif). Apple will not work with apple.gif.

  2. How do you add an item?

    From Admin -> Points System -> Shop: Configuration update the Shop you wish to add the Item to.

  3. Does removing a shop delete all the items from the users who own them?

    Yes, if your forum is rather large, it's probably not a good idea to do this while it's busy as it does take quite a few large loops to complete.

  4. What does Restocking do?

    This setting increases the Item stock until it reaches the maximum amount, every amount of time specified by an Admin.

  5. What does Multiple Item Buying do?

    This setting allows your users to buy more than one of each Item.

  6. What does Viewtopic Type do?

    This is a setting to allow you to disable images while viewing topics. It's mainly there so users with huge amounts of items don't flood your forum's screens. 'Link' just leaves the link to their "Inventory", 'Images' leaves the link and images.

  7. What does Selling Rate do?

    This setting changes the amount users are given back when returning/selling an Item. This is in percent. 100% is a full refund.

  8. How do I make the Effects Shop work?

    From Admin -> Points System -> Shop: Configuration, click on the Edit button next to Special Shop and enable it.

  9. What is the Access Forum field for?

    It allows you to set up private forums that can only be accessed when a user owns a certain item. Make sure view/post/read/edit are all set to PRIVATE if you want to make use of this feature.
